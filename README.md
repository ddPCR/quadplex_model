# quadplex_model

This repository will hold code pertaining to analyzing QXOne ddPCR data to assess the amount of intact nucleic acid template using a similar binomial distribution modeling approach to Tereshko et al. 2023 (PMID: 38096204)

The repo holds all information related to the QXOne manuscript

An Renv lock file was created using R/4.2.2 with packages used in these codes
